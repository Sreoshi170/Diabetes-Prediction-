

# 🩺 Diabetes Prediction using Machine Learning

## 📌 Overview

This project aims to predict whether a person is likely to have **diabetes** based on medical and diagnostic features using Machine Learning algorithms.

Early prediction of diabetes can help in timely treatment and prevention of serious health complications. This project demonstrates how data-driven models can support healthcare decisions.

---

## 🚀 Features

* 📊 Data preprocessing and cleaning
* 🔍 Exploratory Data Analysis (EDA)
* 🤖 Multiple Machine Learning models
* 🏆 Best model selection
* 📈 Data visualization for insights
* 💾 Trained model saved for future use

## 📊 Dataset

The dataset includes important medical attributes such as:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin Level
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age

### 🎯 Target Variable

* `0` → Non-Diabetic
* `1` → Diabetic
